Программа показывает этажи, на которые приедет лифт поочерёдно, согласно вызовам 
c определённого этажа с помощью функции pickup. Если человеку нужно ехать вниз
он нажмет "-1", если вверх, тогда он выберет "1"

В программе не учитывается фактор времени типа, который можно задать, например

class Time():
    def __init__(self):
        self.time=0
    def add(self,num):
        self.time+=num
        return self.time

поэтому не реализована функция update()

Код и симуляция представлены в файле "lift.py"
